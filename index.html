Tutorial 1 - Block, Inline and Breaks

In HTML Elements are containers for information, typically text or images. Each Element can be categorized as a Block Element or an In-line Element.

  • Block: These elements create a new vertical block, stacking content ontop of each other
  • In-line: These elements create a new horiztonal block, keeping content in a horizontal line with each other.

There are many different Elements in HTML, but with clever use of Block and In-line elements, you can easily structure the layout of your content.

Additionally, you can use the br element inbetween in-line elements to Break them up, causing them to stack like Block elements.

Tutorial 2 - Positioning and Spacing

In CSS Properties are the aspects of Elements we can change and manipulate. By using the Position property, we can manipulate the position of an Element in our HTML.

There are a few different kinds of Position Property

  • Static: This is the default setting.
  • Relative: This allows adjustments based on the default positioning through the use of Top, Bottom, Left, and Right properties.
  • Fixed: This makes an Element stay in one position regardless where the user has scrolled on the page. This tutorial used Position:Fixed for the Side Bar and Header.
  • Absolute: This positions an Element based on the position of another Element that it is nested within that contains Position: Relative. Additionally it can use the Top, Bottom, Left, and Right properties.

Click for More Information on the Position Property

Additionally, we can consider using the CSS properties of Margin and Padding to better control the space of a particular HTML element. Margins add space on the outside of a given HTML element, where as Padding adds space inside the bounds of a given HTML element(essentially making it take up more space).

Tutorial 3 - Visual Hierarchy

What is it?

Before we get into visual hierarchies, let's get a basic understanding of what it really is. Visual Hierarchy refers to a methodology that organizes design elements by importance with the aim of drawing attention to the focal points you want users to interact with.

How to apply Visual Hierarchy to your content?

For our purposes, we will focus on the following areas as it relates to creating visual hierarchies

  • Size and Scale
  • Typography
  • Color and contrast
  • White spacing

Size and Scale

We can use size and scale to indicate which information should be prioritized.

Typography

We can make use of different typefaces to emphasize certain pieces of text. Content that we want to highlight should appear larger or bolder while supplementary text should be smaller and use a less dominant font.

Color and Contrast

The use of color gives us another way to draw a user's attentiion to content we want them to look at first.

White Spacing

The use of space helps with legibility and balances out visual hierarchy by preventing too many elements from cluttering the layout. We can make use of margins and line height to accomplish this.
